MySQL 及其變體（包括 MariaDB 和 Percona）廣泛用於許多企業應用程式。這些應用程式包括全球社交網站、大型電子商務系統以及包含數千個資料庫執行個體的 SMB 託管系統。本文檔介紹了配置要求，並提供了在NetApp ONTAP資料管理軟體上部署 MySQL 的調整和儲存配置指導。若要確定本報告中指定的環境、配置和版本是否支援您的環境，請查閱互通性矩陣工具 (IMT)。
