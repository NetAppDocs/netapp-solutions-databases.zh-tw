[.underline]* ONTAP和 CVO 設定*

*配置並啟動作業範本。 *

. 建立作業模板。
+
.. 導覽至資源 → 範本 → 新增，然後按一下新增作業範本。
.. 輸入名稱ONTAP/CVO 設定
.. 選擇作業類型；運行根據劇本配置系統。
.. 為劇本選擇相應的庫存、項目、劇本和憑證。
.. 為 On-Prem 環境選擇 ontap_setup.yml 劇本，或選擇 cvo_setup.yml 複製到 CVO 實例。
.. 將從步驟 4 複製的全域變數貼上到 YAML 標籤下的範本變數欄位中。
.. 按一下「Save（儲存）」。


. 啟動作業模板。
+
.. 導航至資源 → 範本。
.. 按一下所需的模板，然後按一下啟動。
+

NOTE: 我們將使用此模板並將其複製出來用於其他劇本。
