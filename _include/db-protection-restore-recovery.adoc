[.underline]*排程日誌複製劇本*

*配置並啟動作業範本。 *

. 複製先前建立的作業範本。
+
.. 導航至資源 → 範本。
.. 找到ONTAP/CVO 設定模板，然後在最右側點擊“複製模板”
.. 在複製的範本上按一下“編輯範本”，並將名稱變更為“還原和復原手冊”。
.. 為範本保留相同的庫存、項目和憑證。
.. 選擇 ora_recovery.yml 作為要執行的劇本。
.. 變數將保持不變，但需要在變數 dst_cluster_ip 中設定 CVO 叢集 IP。
.. 按一下「Save（儲存）」。


+

NOTE: 直到您準備好在遠端站點恢復資料庫時，才會執行此劇本。
